AnnealMail post-quantum encryption for your email

named for the physical process used by some quantum computers: quantum annealing

start using smarter encryption today in your browser

What makes AnnealMail special?

  • GPG/PGP encryption is replaced by CodeCrypt
  • gpg's factorization and elliptical curves are replaced by algorithms believed to be much more resilient to quantum computers
  • even before quantum computers are developed, you can start encrypting your communication so it's difficult to decrypt later

Steps

  1. Hide any features other than signing and verifying messages
  2. Create a replacement for gpg-agent, to manage keys
  3. Write documentation to install CodeCrypt and use the plugin, including information about quantum computing.
  4. Release a beta plugin for signing and verifying messages using CodeCrypt
  5. Adapt Enigmail's translations and documentation for Annealmail

An open source experiment

This plugin is based on the popular and user-tested Enigmail plugin for encrypting mail in Thunderbird. Changing over to AnnealMail for available options should be simple and secure.

Very few people have used AnnealMail's underlying encryption and decryption, so it's best to keep Enigmail and use message-signing unless your contact recommends that you both use AnnealMail.

This project is not authorized by the developers of Enigmail or CodeCrypt. Use at your own risk.
